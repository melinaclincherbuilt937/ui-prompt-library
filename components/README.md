## How to Add a Component Prompt

1. Copy [`../templates/PROMPT_TEMPLATE.md`](../templates/PROMPT_TEMPLATE.md)
2. Scope it to a **single component** — not a full page
3. Include exact Tailwind classes, animation values, and any special CSS
4. Submit a PR — see [`../CONTRIBUTING.md`](../CONTRIBUTING.md)
